Маркированный список

Тег <ul> устанавливает маркированный список. Каждый элемент списка должен начинаться с тега <li>. Если к тегу <ul> применяется таблица стилей, то элементы <li> наследуют эти свойства.

Атрибуты:

1) type - устанавливает вид маркера списка (disk, circle, square)
2) Доступны универсальные атрибуты*

Пример кода:
<ul>
<li>Чебурашка</li>
<li>Крокодил Гена</li>
<li>Шапокляк</li>
</ul>

Результат:

  • Чебурашка
  • Крокодил Гена
  • Шапокляк

Нумерованный список

Эти списки представляют собой набор элементов с их порядковыми номерами. Вид и тип нумерации зависит от атрибутов тега <ol>, который и применяется для создания списка.

Атрибуты:

1) type - устанавливает вид маркера списка.
2) reserved - нумерация списка становится по убыванию.
3) start - задает число, с которого будет начинаться нумерованный список.
4) Доступны универсальные атрибуты*

Пример кода:
<ol start="3" reversed>
<li>Чебурашка</li>
<li>Крокодил Гена</li>
<li>Шапокляк</li>
</ol>

Результат:

  1. Чебурашка
  2. Крокодил Гена
  3. Шапокляк

Список определений

Список определений состоит из двух элементов — термина и его определения. Сам список задается с помощью контейнера <dl>, термин — тегом <dt>, а его определение — с помощью тега <dd>.

Атрибуты:

1) Доступны универсальные атрибуты*

Пример кода:
<dl>
<dt>GIF</dt>
<dd>Формат графических файлов, широко применяемый при создании сайтов.
GIF использует 8-битовый цвет и эффективно сжимает сплошные цветные области,
при этом сохраняя детали изображения.</dd>
<dt>JPEG</dt>
<dd>Популярный формат графических файлов, широко применямый при создании
сайтов и хранения изображений. JPEG поддерживает 24-битовый цвет и сохраняет
яркость и оттенки цветов в фотографиях. Данный формат называют сжатием с потерями,
поскольку алгоритм JPEG выборочно отвергает данные. Метод сжатия может исказить
деталь в рисунке, особенно содержащий текст или изображение с четкими краями.
Формат JPEG не поддерживает прозрачность, когда вы сохраняете фотографию в формате
JPEG, прозрачные пиксели заполняются определенным цветом.</dd>
</dl>

Результат:

GIF
Формат графических файлов, широко применяемый при создании сайтов. GIF использует 8-битовый цвет и эффективно сжимает сплошные цветные области, при этом сохраняя детали изображения.
JPEG
Популярный формат графических файлов, широко применямый при создании сайтов и хранения изображений. JPEG поддерживает 24-битовый цвет и сохраняет яркость и оттенки цветов в фотографиях. Данный формат называют сжатием с потерями, поскольку алгоритм JPEG выборочно отвергает данные. Метод сжатия может исказить деталь в рисунке, особенно содержащий текст или изображение с четкими краями. Формат JPEG не поддерживает прозрачность, когда вы сохраняете фотографию в формате JPEG, прозрачные пиксели заполняются определенным цветом.

Вложенный список

Вложенные списки предназначены для организации сложной иерархической структуры текста, обычно таких документов, как юридические или технические. Также вложенные списки используются для создания многоуровневых меню и навигации по сайту. Браузеры самостоятельно меняют вид маркеров вложенного списка, чтобы они различались от родительского списка. Исключением является Opera, в которой маркеры не меняются.

Пример кода:
<ul>
 <li>Маша</li>
 <li>Саша</li>
 <ul>
  <li>Лена</li>
  <li>Катя</li>
  <ul>
   <li>Таня</li>
   <li>Ваня</li>
   <ol>
    <li>Миша</li>
    <li>Гриша</li>
   </ol>
  </ul>
 </ul>
</ul>

Результат:

  • Маша
  • Саша
    • Лена
    • Катя
      • Таня
      • Ваня
        1. Миша
        2. Гриша
