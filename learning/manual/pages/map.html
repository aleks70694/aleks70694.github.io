Тег <map>

Тег <map> служит контейнером для элементов <area>, которые определяют активные области для карт-изображений. Такие области устанавливают невидимые зоны на изображении, являющиеся ссылками на HTML-документы. Цель использования тега <map> — в связывании тега <img> с клиентской картой-изображением. Эта связь определяется применением единого идентификатора как в теге <img>, задаваемого атрибутом usemap, так и в теге <map>, устанавливаемого атрибутом name.

Атрибуты:

  • name - имя карты изображений

Тег <area>

Каждый элемент <area> определяет активные области изображения, которые являются ссылками. Рисунок с привязанными к нему активными областями называется в совокупности картой-изображением. Такая карта по внешнему виду ничем не отличается от обычного изображения, но при этом оно может быть разбито на невидимые зоны разной формы, где каждая из областей служит ссылкой. Тег <area> задает форму области, ее размеры, устанавливает адрес документа, на который следует сделать ссылку, а также имя окна или фрейма, куда браузер будет загружать документ. Этот тег всегда располагается в контейнере <map>, который связывает координаты областей с изображением.

Несколько областей могут перекрывать друг друга, сверху будет та, которая в коде HTML располагается выше.

Атрибуты:

  • accesskey - переход к области с помощью комбинации клавиш.
  • coords - устанавливает координаты активной области.
  • alt - альтернативный текст для области изображения.
  • href - задает адрес документа, на который следует перейти.
  • hreflang - указывает язык документа, на который ведет ссылка.
  • nohref - область без ссылки на другой документ.
  • shape - форма области.
  • tabindex - определяет последовательность перехода между ссылками при нажатии на кнопку Tab.
  • target - имя окна или фрейма, куда браузер будет загружать документ.
  • type - указывает MIME-тип документа, на который ведёт ссылка.
  • Доступны универсальные атрибуты*

Пример кода:
<p><map name="minion">
<area shape="rect" coords="25,25,125,125" href="https://goo.gl/TJM03p" alt="Квадрат" />
<area shape="circle" coords="200,75,50" href="https://goo.gl/McMWR0" alt="Круг" />
<area shape="poly" coords="260,125,390,125,325,25" href="https://goo.gl/sIzlKj" alt="Треугольник" />
<area shape="poly" coords="400,75,435,57,450,22,465,60,500,75,465,90,450,125,435,90" href="https://goo.gl/8fn1kV" alt="Звезда" />
</map></p>
<p><img src="images/map.png" usemap="#minion" alt="Пример карты"></p>
