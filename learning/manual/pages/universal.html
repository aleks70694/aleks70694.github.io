Универсальные атрибуты применяются практически ко всем тегам, поэтому выделены в отдельную группу, чтобы не повторять их для всех тегов.

Атрибут Описание Значения
accesskey Позволяет получить доступ к элементу с помощью заданного сочетания клавиш. Цифра (0-9) или латинская буква (a-z).
class Определяет имя класса, которое позволяет связать тег со стилевым оформлением. Значения по умолчанию нет, придумывается разработчиком.
contenteditable Сообщает, что элемент доступен для редактирования пользователем.
  • true - включает режим редактирования.
  • false - запрещает редактирование элемента.
contextmenu Устанавливает контекстное меню для элемента. В качестве значения указывается идентификатор меню, созданного с помощью тега <menu>.
dir Задает направление и отображение текста — слева направо или справа налево.
  • ltr - текст отображается как обычно — слева направо.
  • rtl - текст инвертируется и отображается справа налево.
hidden Скрывает содержимое элемента от просмотра. Можно оставить пустым.
id Указывает имя стилевого идентификатора. Значения по умолчанию нет, придумывается разработчиком.
lang Браузер использует значение параметра для правильного отображения некоторых национальных символов. Коды языков.
spellcheck Указывает браузеру проверять или нет правописание и грамматику в тексте.
  • true - включает режим проверки правописания.
  • false - отключает проверку.
style Применяется для определения стиля элемента с помощью правил CSS. В качестве значений указываются стилевые правила: вначале следует имя стилевого свойства, затем через двоеточие его значение. Стилевые свойства разделяются между собой точкой с запятой.
tabindex Устанавливает порядок получения фокуса при переходе между элементами с помощью клавиши Tab. Любое целое положительное число.
title Описывает содержимое элемента в виде всплывающей подсказки. Любая текстовая строка. Строка должна заключаться в двойные или одинарные кавычки.
xml:lang Этот атрибут по своему действию похож на lang, но применяется только в XHTML-документах и указывает язык всего текста или его фрагмента. Коды языков.
